Projects

Conference Client

This project was completed in a group of 7 students as the final project for CSC207: Software Design. We made a conference client in Java with an interactive Java Swing GUI using best practices like SOLID architecture and various design patterns like the MVP, Factory, Builder and Facade design patterns.

Musical Neural Style Converter

Led a team of 5 developers in creating a software program that uses a deep learing model to transfer the "style" of an input song to that of the target song using Python’s librosa and Tensorflow libraries.

DoodleJump Game (Assembly)

Made a scaled down version of the popular doodle jump game in Assembly langage on MARS MIPS emulator. This project was the final project for CSC258: Computer Organisation. I didn't stop at just completing the requirements and added better graphics witha wallpaper, enemies and improved game physics.

Moods Journal

Moods is a Journal app that uses sentiment analysis to detect mood levels of the user and makes it easier for them to keep track of their mental health. The app also displays useful statistics that help users identify what factors affected their mental health the most over time (positively and negatively). The front end was implemented using flutter and the backend was implemented using Python.

Machine Learning Algos in C

As assigments for the course CSC209: Software tools and systems programming I implemented KNNs and Decision Trees in C; used the concepts of pipes and processes to speed up the timimg of the algorithms and get better results.
Note: I was provided starter code by the course instructor, but the important parts like classification and forking were implemented by me. Access to github repo can be provided if needed.

Snake Game

Worked on making the classic snake game in javascript to get experience in working with the laguage and practicing my HTMl and CSS skills. The game is actually very fun to play at a high speed setting and has a custom 8-bit background song.
